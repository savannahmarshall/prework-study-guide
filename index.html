HTML

  • HTML gives the content structure.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1 to h6 elements are used as headings and make the page accessible to all users. The alt attribute describes the image. It can improve the search engine rankings for your website because it is used by screen readers and assistive technologies.
  • The h1 to h6 headings need to be in order and h1 should only appear once on each page.
  • The title element defines the title for a webpage.
  • The base defines the URL for a webpage.
  • The meta defines the metadata such as character set, description, key words, author and viewpoint.

CSS

  • CSS gives the content style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations contain two components: The CSS propety we want to apply and the value of the property.
  • We use a * symbol to apply a rule to all elements visbile on the page and declared that all elemenets will ahve the same style properies margin and padding set at 0.
  • We use the HTML link element to link the CSS file to the HTML file.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout main: switches to the main branch.
  • git add -A: commits all files in the working branch
  • git commit -m: use the -m flag to associate a message with our commit
  • git push origin feature/add-html: pushes up all of our changes

JavaScript

  • JavaScript is used to handle logic and allows users to interact with a website.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a set of instructions that tells the computer how to perform a certain task and a function's name is followed by parentheses.
  • A for loop contains the starting point for our loop, a condition and the final statement.
